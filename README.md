## How to use it?

Save your data in `json` format like this:

```
{
	"samples": [
		[1, 3],
		[1, 4],
		[2, 4],
		[3, 1],
		[4, 1],
		[4, 2]
	],
	"labels": ["a", "a", "a", "b", "b", "b"]
}

```

After that upload your file to a server & enjoy it)
